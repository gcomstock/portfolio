import imgBroadNav from './images/broad-nav.gif';
import imgBroadIA  from './images/broad-ia.gif';

<ImagePanel
  image={imgBroadNav}
  alt="Early wireframe defining page layouts and UX patterns for Spotlight"
  title="Defining Page Layouts and UX Patterns"
>
  <p>Because the project was new, there were no existing patterns to inherit. I started with low-fidelity Figma mocks to reach alignment, then designed a set of higher-fidelity exemplar plugins that became the reference standard for future plugin providers. Those patterns eventually lived in both a Figma component library and the codebase itself, with the goal that providers could make sound design decisions without needing me in the room.</p>
  <p>One point of contention was the detail panel: a right-side drawer that opened when a user clicked on an entity. Critics felt it obscured the main content area. The counter-argument was that once a user opened a detail panel, it became their primary area of focus. Keyboard shortcuts let them move between entities without ever returning to the list, making the main content area a secondary concern in that moment. And for larger viewports, a max-width constraint on the main content column meant the panel opened beside the content rather than over it.</p>
</ImagePanel>

<ImagePanel
  image={imgBroadIA}
  alt="Early wireframe establishing the information architecture for Spotlight"
  title="Establishing the Information Architecture"
  reverse
>
  <p>Existing developer tools at Netflix were inconsistent. Some let you view specific applications or teams, some didn't. Some let you save entities, some didn't. The IA work aimed to bring enough consistency that any plugin provider could develop against it and have their experience feel seamless alongside every other plugin.</p>
  <p>We landed on a sidebar for top-level plugin navigation, keeping traversal simple regardless of how many plugins existed. Within a plugin, breadcrumbs handled deeper navigation, but switching plugins reset the breadcrumb trail. A global search tied it all together, letting users jump directly to an application, team, or collection and have the sidebar reflect their context automatically. The implicit promise to plugin providers: build inside Spotlight, and all of this comes for free.</p>
</ImagePanel>
